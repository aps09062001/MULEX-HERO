# MULEX-HERO

## Brand Book - Manual de Marca

Este repositorio contiene el Brand Book completo de MULEX HERO.

### Cómo usar tu imagen del anuncio de prensa

El archivo `index.html` ahora está configurado para usar una imagen local llamada `ANUNCIO HERO.png` en la sección de prensa.

**Para reemplazar la imagen placeholder con tu imagen real:**

1. Reemplaza el archivo `ANUNCIO HERO.png` en la raíz del proyecto con tu imagen final
2. Asegúrate de que el archivo mantenga el mismo nombre: `ANUNCIO HERO.png`
3. Abre `index.html` en tu navegador para ver el resultado

**Nota:** La imagen placeholder actual tiene dimensiones de 2070x1035 píxeles. Para mejores resultados, usa una imagen con dimensiones similares (aspect ratio 2:1).

### Visualización

Para ver el Brand Book, simplemente abre el archivo `index.html` en tu navegador web.
